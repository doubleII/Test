## Commands

**choco help info**

```bash
choco --help
```

**create choco package**

```bash
choco pack
```

**install choco package**

```bash
choco install <package_name> [<options/switches>]
```

**options**

`-y` - yes, confirm all prompts

`-s` - install sidy by side multiple version of packages
