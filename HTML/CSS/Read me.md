### Tips

1. Write two words on one line 

`word1$nmsp;word2`

2. Absolute urls

`<body> <a href="/index.html">Home Page</a> </body>`

3. Relative urls

`<body> <a href="../index.html">Home Page</a> </body>`</br>

../ - one level higher </br>

../../../ - tree level higher </br>
