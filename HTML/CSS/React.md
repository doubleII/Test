### Tips

##### How to write two words they always should be on one line 
`word1$nmsp;word2`</br>

##### Absolute urls
`<body> <a href="/index.html">Home Page</a> </body>`</br>

##### Relative urls
`<body> <a href="../index.html">Home Page</a> </body>`</br>
../ - one level higher </br>
../../../ - tree level higher </br>
